# ModPathfinder

Simulator um den Driver Mechanismus zu testen.

## Einsatz
In `Robot` ist der Drive implementiert. 
Es können mehrere Roboter fahren.

## Tastaturbelegung
- `space`: Start/ Stopp
- `A`: Neuen Roboter hinzufügen
- `R`: Reload
- `X`: Debug Case 1
- `Y`: Debug Case 2
- `Up`: Schneller
- `Down`: Langsamer
